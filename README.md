# Sistema_Estoque

# 📋 Requisitos Funcionais - Sistema de Controle de Estoque

| Código | Requisito | Tipo Funcional | Descrição | Prioridade |
|--------|----------|----------------|-----------|------------|
| RF01 | Cadastro de funcionários | Cadastro | O usuário deve ser capaz de registrar funcionários | Alta |
| RF02 | Controle de funcionários | Controle | O usuário deve ser capaz de modificar os dados dos funcionários | Alta |
| RF03 | Cadastro de produto | Cadastro | O usuário deve ser capaz de cadastrar um novo produto | Alta |
| RF04 | Controle de produtos | Controle | O usuário deve ser capaz de modificar os dados dos produtos | Alta |
| RF05 | Compras | Operação | O sistema deve registrar as compras feitas | Alta |
| RF06 | Vendas | Operação | O sistema deve registrar as vendas feitas | Alta |
| RF07 | Consultar inventário | Consulta | O usuário deve ser capaz de checar o inventário | Alta |
| RF08 | Controle de compras | Controle | O sistema deve atrelar uma compra/pedido a um funcionário e um fornecedor | Alta |
| RF09 | Controle de vendas | Controle | O sistema deve atrelar uma venda a um funcionário e um cliente | Alta |
| RF10 | Cadastro de categoria | Cadastro | Usuário deve conseguir registrar uma nova categoria de produto | Alta |
| RF11 | Controle de categoria | Controle | Usuário deve conseguir modificar as categorias de produtos | Alta |
| RF12 | Cadastro de fornecedor | Cadastro | Usuário deve ser capaz de cadastrar um fornecedor | Alta |
| RF13 | Controle de fornecedor | Controle | Usuário deve ser capaz de modificar os dados de um fornecedor | Alta |
| RF14 | Cadastro de cliente | Cadastro | Usuário deve ser capaz de cadastrar clientes | Média |
| RF15 | Controle de cliente | Controle | Usuário deve ser capaz de modificar os dados de um cliente | Média |
| RF16 | Login | Autenticação | Sistema deve permitir autenticação de usuários | Alta |
| RF17 | Baixo estoque | Alerta | Sistema deve alertar baixo estoque | Alta |
